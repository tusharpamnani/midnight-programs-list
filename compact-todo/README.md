# 📝 Todo Compact on Midnight 

A privacy-preserving Todo application built on the **Midnight Network** using the Compact language.

This project demonstrates how to build **confidential smart contracts** where:
- Todo content remains private  
- Ownership is cryptographically enforced  
- State transitions are publicly verifiable  

---
